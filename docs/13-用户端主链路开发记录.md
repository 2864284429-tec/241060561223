### 3. 实现详情

#### 3.1 HomeController.Index

```csharp
public IActionResult Index()
{
    return View();
}
```

- 移除了 `ErrorViewModel` 和 `Activity` 依赖
- 视图显示欢迎语、当前日期、快捷入口（座位列表 / 我的预约）、预约须知

#### 3.2 SeatController

```csharp
public class SeatController : Controller
{
    private readonly ISeatService _seatService;
    // List → GetAllSeatsWithStatus()
    // Detail(int id) → GetSeatDetail(id), null → NotFound()
}
```

- 路由：`/Seat/List`、`/Seat/Detail/{id}`（与 docs/09 §2.2 一致）
- 注入 ISeatService，调用 Sprint 0 已实现的方法

#### 3.3 座位列表页

- 使用 `@model List<SeatListItem>` 接收数据
- 动态计算可用座位数，显示提示横幅
- 每个座位卡片：名称、位置、Features、状态标签
- 不可用/已预约座位降低透明度（opacity: 0.5）
- 空列表显示空状态组件
- 底部状态标签图例

#### 3.4 座位详情页

- 使用 `@model SeatDetailViewModel` 接收数据
- 显示座位名称、位置、Features、容量、状态
- 日期选择器（今天 + 未来 4 天，共 5 天）
- 时段选择网格（8:00-20:00，共 13 个时段）
- 确认预约按钮（可用时显示，不可用时禁用）
- 返回座位列表按钮

#### 3.5 _Layout.cshtml

- 简化为最小结构：DOCTYPE → head → body → @RenderBody
- 移除了默认 Bootstrap 导航栏和 Footer
- 每个用户页面自行实现 navbar（匹配原型设计）
- lang 改为 zh-CN，标题改为中文

#### 3.6 自定义样式

- 完整迁移 `prototype/static-v1/css/custom.css`
- 包含：设计令牌、导航栏、卡片、状态标签、按钮、输入框、表格、时段选择、进度条、空状态、响应式等
- 适配用户端和管理端
