### C2：优化 GetAllSeatsWithStatus() 查询

**问题**：`Select()` 内嵌套子查询会产生 N+1 问题。

**修复方案**：改为两次查询或使用 `GroupJoin`：

```csharp
public List<SeatListItem> GetAllSeatsWithStatus()
{
    var today = DateTime.Today;
    var currentSlot = DateTime.Now.Hour;

    // 一次查询获取所有座位
    var seats = _dbContext.Seats.ToList();

    // 一次查询获取当前时段所有预约
    var reservedSeatIds = _dbContext.Reservations
        .Where(r => r.Date == today 
                  && r.TimeSlot == currentSlot 
                  && r.Status == "已预约")
        .Select(r => r.SeatId)
        .ToHashSet();

    return seats.Select(seat => new SeatListItem
    {
        Id = seat.Id,
        Name = seat.Name,
        Location = seat.Location,
        Device = seat.Device,
        IsAvailable = seat.IsAvailable,
        CurrentStatus = !seat.IsAvailable ? "不可用" :
            reservedSeatIds.Contains(seat.Id) ? "已预约" : "可用"
    }).ToList();
}
```

## D. 建议优化项

### S1：CancelReservation 返回值优化

**问题**：返回 `bool` 无法告知用户失败原因。

**建议**：改为返回 `ServiceResult`：

```csharp
public ServiceResult CancelReservation(int reservationId, string userId)
{
    var reservation = _dbContext.Reservations
        .FirstOrDefault(r => r.Id == reservationId && r.UserId == userId);

    if (reservation == null)
        return ServiceResult.Fail("预约不存在");

    if (reservation.Status != "已预约")
        return ServiceResult.Fail("预约状态异常");

    var slotEndTime = reservation.Date.AddHours(reservation.TimeSlot + 1);
    if (DateTime.Now >= slotEndTime)
        return ServiceResult.Fail("已过时，无法取消");

    reservation.Status = "已取消";
    reservation.UpdatedAt = DateTime.UtcNow;
    _dbContext.SaveChanges();

    return ServiceResult.Success();
}
```

Controller 处理：

```csharp
[HttpPost]
public IActionResult Cancel(int id)
{
    var userId = HttpContext.Session.GetString("CurrentUserId");
    var result = _reservationService.CancelReservation(id, userId);

    if (result.IsSuccess)
        TempData["SuccessMessage"] = "取消成功";
    else
        TempData["ErrorMessage"] = result.Message;

    return RedirectToAction("My");
}
```

### S5：补充 UpdatedAt 自动更新机制

**问题**：§6.1 补足内容未说明 `UpdatedAt` 如何自动更新。

**建议**：补充 EF Core 配置：

```csharp
// AppDbContext.cs
protected override void OnModelCreating(ModelBuilder modelBuilder)
{
    modelBuilder.Entity<Seat>()
        .Property(s => s.UpdatedAt)
        .ValueGeneratedOnAddOrUpdate();

    modelBuilder.Entity<Reservation>()
        .Property(r => r.UpdatedAt)
        .ValueGeneratedOnAddOrUpdate();
}
```

或在 `SaveChanges` 中手动设置：

```csharp
public override int SaveChanges()
{
    foreach (var entry in ChangeTracker.Entries()
        .Where(e => e.State == EntityState.Modified))
    {
        if (entry.Entity is Seat seat)
            seat.UpdatedAt = DateTime.UtcNow;
        else if (entry.Entity is Reservation reservation)
            reservation.UpdatedAt = DateTime.UtcNow;
    }
    return base.SaveChanges();
}
```
