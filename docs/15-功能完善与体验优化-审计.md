## C. 必改项

### B-01: AdminSeatController 补充异常处理

**当前状态：** 三个 POST Action 均无 try/catch，Service 层异常直接穿透到框架层。

**建议修改：** 在三个 POST Action 中添加 try/catch，捕获异常时设置 ErrorMessage TempData 并 redirect。

```csharp
// 示例：Create Action
try
{
    var success = _seatService.CreateSeat(model);
    TempData[success ? "SuccessMessage" : "ErrorMessage"] =
        success ? "座位新增成功" : "座位新增失败，请重试";
}
catch (Exception ex)
{
    TempData["ErrorMessage"] = "系统异常，请稍后重试";
    // 可选：记录日志 _logger.LogError(ex, "...");
}
```

**影响范围：** AdminSeatController.cs Create/Edit/Delete 三个方法。
