## C. 必改项（详细说明）

### 问题 1：统计页侧边栏死链

**现状：** `_AdminLayout.cshtml` 第 50 行：
```html
<a asp-controller="Admin" asp-action="Stats" class="admin-sidebar-item ...">
```
但 `AdminController.cs` 无 `Stats` 方法。管理员点击「统计」→ 404。

**根因：** Sprint 3 编写 `_AdminLayout.cshtml` 时预设了统计页，但 T14-01/T14-02（统计页）尚未开发。

**修复建议（二选一）：**
- **方案 A（推荐）**：在 `AdminController` 中添加 `Stats` action 最小实现（返回空统计页或占位视图），使链接不 404
- **方案 B**：从 `_AdminLayout.cshtml` 侧边栏移除「统计」链接，待 T14-01 开发时再加回

### 问题 2：导航高亮逻辑错误

**现状：** 第 5 行 `isActiveStats = currentAction == "Index"` — 比较对象错误。`Index` 是 `HomeController` 的 action，`AdminController` 中无此 action。

**修复建议：** 改为 `currentAction == "Stats"`。
