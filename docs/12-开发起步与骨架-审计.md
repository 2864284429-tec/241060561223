## C. 必改项

> 以下问题影响文档准确性，建议在 Sprint 1 开始前修正。

### R-01：README 种子数据初始化描述与代码不符

**位置**：README.md 第 109 行

**当前内容**：
```
> 数据库建表和种子数据在首次启动时通过 `Program.cs` 中的 `db.Database.Migrate()` + `SeedData.Initialize()` 自动完成。
```

**问题**：代码中不存在 `SeedData.Initialize()` 方法。种子数据通过 EF Core `HasData()` 在 `OnModelCreating` 中定义，由 `Migrate()` 自动应用。`SeedData.Initialize()` 是 docs/08 §10.3 的示例代码，不是实际实现。

**建议修改为**：
```
> 数据库建表和种子数据在首次启动时通过 `Program.cs` 中的 `db.Database.Migrate()` 自动完成（种子数据通过 EF Core `HasData()` 定义在 `AppDbContext.OnModelCreating` 中）。
```

### R-02：README 解决方案文件名与实际不符

**位置**：README.md 第 65 行目录结构

**当前内容**：
```
├── LibrarySeatReservation.sln
```

**实际文件**：`LibrarySeatReservation.slnx`（.NET 10 新格式）

**建议修改为**：
```
├── LibrarySeatReservation.slnx
```
