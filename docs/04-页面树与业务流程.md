## 2. 用户端页面树

```
用户首页（/Home/Index）
  │
  ├──→ 座位列表页（/Seat/List）
  │       │
  │       └──→ 座位详情页（/Seat/Detail/{id}）
  │               │
  │               └──→ 预约提交页（/Reservation/Create）
  │                       │
  │                       └──→ 我的预约页（/Reservation/My）
  │
  └──→ 我的预约页（/Reservation/My）
          │
          └──→ [模态框] 取消预约确认
```

### 2.1 用户首页（/Home/Index）

| 属性 | 内容 |
|------|------|
| 页面类型 | 静态展示页 |
| 主要内容 | 欢迎信息、当前日期、快捷入口按钮 |
| 跳转出口 | → 座位列表页、→ 我的预约页 |

### 2.2 座位列表页（/Seat/List）

| 属性 | 内容 |
|------|------|
| 页面类型 | 列表页 |
| 主要内容 | 座位卡片列表（名称、位置、状态标签） |
| 跳转入口 | ← 用户首页 |
| 跳转出口 | → 座位详情页 |

### 2.3 座位详情页（/Seat/Detail/{id}）

| 属性 | 内容 |
|------|------|
| 页面类型 | 详情页 |
| 主要内容 | 座位详细信息（名称、位置、容量、设备）+ 预约按钮 |
| 跳转入口 | ← 座位列表页 |
| 跳转出口 | → 预约提交页 |

### 2.4 预约提交页（/Reservation/Create）

| 属性 | 内容 |
|------|------|
| 页面类型 | 表单页 |
| 主要内容 | 座位信息摘要、日期选择器、时间段选择、提交按钮 |
| 跳转入口 | ← 座位详情页 |
| 跳转出口 | → 我的预约页（预约成功后跳转） |

### 2.5 我的预约页（/Reservation/My）

| 属性 | 内容 |
|------|------|
| 页面类型 | 列表页 |
| 主要内容 | 预约记录表格（座位、日期、时间段、状态、操作） |
| 跳转入口 | ← 用户首页、← 预约提交页 |
| 跳转出口 | → [模态框] 取消预约确认 |

## 4. 用户主链路

### 4.1 预约主链路（闭环）

```
用户首页
  │ 点击"浏览座位"
  ↓
座位列表页
  │ 选择一个可用座位
  ↓
座位详情页
  │ 点击"预约"
  ↓
预约提交页
  │ 选择日期和时间段
  │ 点击"确认提交"
  ↓
预约成功提示
  │ 自动跳转
  ↓
我的预约页 ← 显示新预约记录
```

**闭环验证**：预约成功后，我的预约页中能看到新记录，状态为"已预约"。

### 4.2 取消主链路（闭环）

```
我的预约页
  │ 找到要取消的记录
  │ 点击"取消"
  ↓
[模态框] 取消预约确认
  │ 显示预约信息摘要
  │ 点击"确认取消"
  ↓
取消成功提示
  │ 关闭模态框
  ↓
我的预约页 ← 记录状态更新为"已取消"
```

**闭环验证**：取消成功后，记录状态变为"已取消"，座位恢复可用。

## 6. 状态流转

### 6.1 预约状态流转

```
[创建预约]
    ↓
  已预约
    ↓
    ├── 用户取消 → 已取消
    ↓
    └── 时间过期 → 已过时（查询时动态判断）
```

| 当前状态 | 触发条件 | 目标状态 | 涉及页面 |
|----------|----------|----------|----------|
| （无） | 提交预约 | 已预约 | 预约提交页 → 我的预约页 |
| 已预约 | 用户点击取消 | 已取消 | 我的预约页（模态框） |
| 已预约 | 当前时间 > 预约结束时间 | 已过时 | 我的预约页（查询时判断） |

### 6.2 座位状态流转

```
[管理员创建座位]
    ↓
  可用
    ↓
    ├── 用户预约该时段 → 已预约（该时段）
    ├── 用户取消 → 可用（该时段）
    ├── 管理员删除 → （记录删除）
    └── 管理员设为不可用 → 不可用
```

| 当前状态 | 触发条件 | 目标状态 | 涉及页面 |
|----------|----------|----------|----------|
| （无） | 管理员新增 | 可用 | 座位管理页 |
| 可用 | 用户预约该时段 | 已预约（该时段） | 预约提交页 |
| 已预约（该时段） | 用户取消 | 可用（该时段） | 我的预约页 |
| 可用 | 管理员删除 | （记录删除） | 座位管理页 |
| 可用 | 管理员编辑 IsAvailable=false | 不可用 | 座位管理页 |

### 6.3 状态与页面对应关系

| 状态变更 | 触发页面 | 结果页面 |
|----------|----------|----------|
| 创建预约（已预约） | 预约提交页 | 我的预约页 |
| 取消预约（已取消） | 我的预约页（模态框） | 我的预约页 |
| 新增座位（可用） | 座位管理页（模态框） | 座位管理页 |
| 编辑座位 | 座位管理页（模态框） | 座位管理页 |
| 删除座位 | 座位管理页（确认框） | 座位管理页 |
