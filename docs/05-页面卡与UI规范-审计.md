## C. 必改项

以下问题会影响静态原型生成的准确性，**建议在进入下一步前修正**：

### C1. 补充页面加载状态描述

在每个页面卡中补充"加载状态"：

| 页面 | 加载状态描述 |
|------|--------------|
| 用户首页 | 无加载状态（静态页面） |
| 座位列表页 | 加载中显示骨架屏或加载图标 |
| 座位详情页 | 加载中显示骨架屏或加载图标 |
| 预约提交页 | 加载中显示骨架屏或加载图标；提交中按钮显示加载状态 |
| 我的预约页 | 加载中显示骨架屏或加载图标 |
| 管理员登录页 | 登录中按钮显示加载状态 |
| 预约管理页 | 加载中显示骨架屏或加载图标 |
| 座位管理页 | 加载中显示骨架屏或加载图标；保存中按钮显示加载状态 |
| 统计页 | 加载中显示骨架屏或加载图标 |

### C2. 明确我的预约页布局方式

在页面卡 5 中明确：

> 手机竖屏：采用**卡片列表**方式展示预约记录（每条记录一张卡片，包含座位名称、日期、时间段、状态标签、取消按钮），不使用横向可滚动表格。
