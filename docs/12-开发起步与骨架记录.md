### 4. 数据库

| 表 | 字段 | 种子数据 |
|----|------|----------|
| Seats | Id, Name, Location, Capacity, Features, IsAvailable, CreatedAt, UpdatedAt | 8 条（A-01~C-02） |
| Reservations | Id, SeatId, UserId, Date, TimeSlot, Status, CreatedAt, UpdatedAt | 0 条（空表） |
| DemoUsers | Id, Name, CreatedAt | 3 条（user1/小王, user2/小李, user3/小张） |

**关键约束：**
- `CK_Reservations_TimeSlot`：TimeSlot >= 8 AND TimeSlot <= 20
- 外键 `Reservations.SeatId → Seats.Id`（RESTRICT）
- 外键 `Reservations.UserId → DemoUsers.Id`（RESTRICT）
- 索引：`IX_Reservations_SeatDateSlot`, `IX_Reservations_UserId_Date`
