## C. 必改项

### C1：补建 .gitignore 和 .gitattributes

**问题**：README 和 docs/10 都列出了这两个文件，但仓库中不存在。

**修复方案**：

1. 创建 `.gitignore`：

```gitignore
# .NET
bin/
obj/
*.user
*.suo
.vs/

# IDE
.vscode/
.idea/

# OS
.DS_Store
Thumbs.db

# Database
*.mdf
*.ldf

# Local
appsettings.Development.json
```

2. 创建 `.gitattributes`：

```
*.cs text diff=csharp
*.cshtml text
*.css text
*.js text
*.md text
*.json text
```

3. 提交：

```bash
git add .gitignore .gitattributes
git commit -m "chore: 添加 .gitignore 和 .gitattributes"
```
