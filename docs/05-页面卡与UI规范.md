## 3. 视觉方向说明

### 3.1 主方向：校园服务感 + 安静学习感

**定义**：系统视觉应该传达"这是一个可靠、安静、高效的校园服务工具"，而不是花哨的商业产品或冰冷的后台系统。

**具体表现**：
- 配色以蓝色为主色（信任感、专业感），绿色为辅助色（可用、成功）
- 圆角适中（8px），不过分圆润也不尖锐
- 间距宽松，留白充足，呼吸感强
- 卡片有轻微阴影，但不浮夸
- 文字清晰易读，层次分明

### 3.2 不是的方向

| 方向 | 为什么不是 |
|------|------------|
| 商业科技感 | 过于炫酷，不适合校园场景 |
| 极简黑白 | 过于冷淡，缺乏亲和力 |
| 活泼多彩 | 过于花哨，干扰信息获取 |
| 复古风格 | 不符合现代 Web 应用习惯 |

---

## 4. 页面氛围关键词与禁用风格

### 4.1 氛围关键词

**核心关键词**：安静、可靠、高效、清晰、友好

**辅助关键词**：校园、学习、秩序、便利

### 4.2 禁用风格

| 禁用项 | 原因 |
|--------|------|
| 渐变背景 | 干扰信息阅读 |
| 闪烁动画 | 分散注意力 |
| 过度阴影 | 显得浮夸 |
| 花哨图标 | 不专业 |
| 低对比度文字 | 可读性差 |
| 过小字体 | 阅读困难 |
| 过密间距 | 压迫感强 |

## 5. 组件规范

### 5.1 按钮

| 类型 | 样式 | 用途 |
|------|------|------|
| 主按钮（Primary） | 背景 `#2563EB`，文字白色，圆角 8px，高度 40px，内边距 0 24px | 主要操作（预约、提交、登录） |
| 次按钮（Secondary） | 背景白色，边框 `#E5E7EB`，文字 `#374151`，圆角 8px，高度 40px | 次要操作（返回、取消） |
| 危险按钮（Danger） | 背景 `#DC2626`，文字白色，圆角 8px，高度 40px | 删除、取消预约 |
| 文字按钮（Text） | 背景透明，文字 `#2563EB`，无边框 | 导航链接、轻量操作 |

**按钮状态**：
- 默认：正常显示
- 悬停：主按钮背景变深（`#1D4ED8`），次按钮边框变深
- 禁用：背景 `#E5E7EB`，文字 `#9CA3AF`，不可点击
- 加载中：显示加载图标，按钮不可点击

## 7. 原型三层次说明

### 7.1 低保真原型

**定义**：线框图，只展示页面结构和信息层级，不涉及视觉细节。

**包含**：
- 页面布局结构
- 信息区块划分
- 按钮位置
- 文字占位

**不包含**：
- 具体颜色
- 字体样式
- 图标细节
- 交互效果

**用途**：验证页面结构和信息架构是否合理。

### 7.2 中保真原型

**定义**：带有基本视觉规范的原型，展示页面的真实外观。

**包含**：
- 低保真的所有内容
- 具体颜色（使用色彩规范）
- 字体样式（使用字体规范）
- 组件样式（使用组件规范）
- 状态标签样式

**不包含**：
- 真实数据
- 完整交互
- 动画效果

**用途**：验证视觉设计是否符合规范，页面是否美观。

### 7.3 高保真原型

**定义**：接近最终产品的原型，包含真实数据和完整交互。

**包含**：
- 中保真的所有内容
- 真实数据
- 完整交互流程
- 动画效果

**用途**：最终验收和开发参考。
