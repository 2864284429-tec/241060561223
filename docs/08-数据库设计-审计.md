## D. 建议优化项

### S1：Status 字段实际写入值说明

**问题**：CHECK 约束允许 `Status IN (N'已预约', N'已取消', N'已过时')`，但"已过时"是查询时动态判断，实际不会写入数据库。

**建议**：在第 6.3 节补充说明：

```markdown
> 说明：数据库中 Status 字段实际只会写入"已预约"和"已取消"两种值。
> "已过时"状态是查询时根据当前时间和时段动态计算的，不会存储在数据库中。
> CHECK 约束保留"已过时"是为了数据完整性，防止历史数据异常。
```

### S2：统计时间范围可配置化

**问题**：第 8.2 节统计 SQL 硬编码了 `DATEADD(DAY, -7, GETUTCDATE())`，即固定统计最近 7 天。

**建议**：补充说明或改为参数化：

```sql
-- 参数化示例
DECLARE @StartDate DATE = DATEADD(DAY, -7, GETUTCDATE());
DECLARE @EndDate DATE = GETUTCDATE();

SELECT 
    s.Name AS SeatName,
    COUNT(r.Id) AS ReservationCount
FROM Seats s
LEFT JOIN Reservations r ON s.Id = r.SeatId
    AND r.Date >= @StartDate
    AND r.Date <= @EndDate
    AND r.Status != N'已取消'
GROUP BY s.Id, s.Name
ORDER BY ReservationCount DESC;
```

### S3：UpdatedAt 自动更新机制

**问题**：Seats 和 Reservations 表都有 `UpdatedAt` 字段，但未说明如何自动更新。

**建议**：补充 EF Core 配置示例：

```csharp
// AppDbContext.cs
protected override void OnModelCreating(ModelBuilder modelBuilder)
{
    modelBuilder.Entity<Seat>()
        .Property(s => s.UpdatedAt)
        .ValueGeneratedOnAddOrUpdate();

    modelBuilder.Entity<Reservation>()
        .Property(r => r.UpdatedAt)
        .ValueGeneratedOnAddOrUpdate();
}

// 或在 SaveChanges 时手动设置
public override int SaveChanges()
{
    foreach (var entry in ChangeTracker.Entries()
        .Where(e => e.Entity is Seat || e.Entity is Reservation))
    {
        if (entry.State == EntityState.Modified)
        {
            entry.CurrentValues["UpdatedAt"] = DateTime.UtcNow;
        }
    }
    return base.SaveChanges();
}
```

### S5：TimeSlot 与实际时间对应关系

**问题**：第 5.1 节列出了 TimeSlot 值与时间的对应关系，但未说明如何在代码中转换。

**建议**：补充转换工具方法：

```csharp
public static class TimeSlotHelper
{
    public const int MinSlot = 8;
    public const int MaxSlot = 20;
    public const int SlotCount = 13; // 8:00-21:00

    /// <summary>
    /// 获取时段开始时间
    /// </summary>
    public static DateTime GetSlotStartTime(DateTime date, int timeSlot)
    {
        return date.AddHours(timeSlot);
    }

    /// <summary>
    /// 获取时段结束时间
    /// </summary>
    public static DateTime GetSlotEndTime(DateTime date, int timeSlot)
    {
        return date.AddHours(timeSlot + 1);
    }

    /// <summary>
    /// 获取所有可用时段列表
    /// </summary>
    public static List<int> GetAvailableTimeSlots()
    {
        return Enumerable.Range(MinSlot, SlotCount).ToList();
    }

    /// <summary>
    /// 判断时段是否已过时
    /// </summary>
    public static bool IsSlotExpired(DateTime date, int timeSlot)
    {
        return DateTime.Now >= GetSlotEndTime(date, timeSlot);
    }
}
```
